Desafio DevOps

Fui desafiado a criar uma aplicação simples com uma página HTML customizável, definida via Helm e hospedada em um cluster Kubernetes. Aqui estão os requisitos principais:

  • Uso de repositório git com Dockerfile e Helm;
  • Imagem construída e publicada no Docker Hub;
  • A imagem não deve rodar com usuário root;
  • A página deve ser configurável via ConfigMap;
  • O Helm define todos os objetos da aplicação;
  • A aplicação possui um domínio para acesso;
  • Todos os objetos no cluster têm a label desafio=jackexperts;
  • Documentação completa da solução.
